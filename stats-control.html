How to Update Stats

  1. Open this HTML file in a text editor (VS Code, Notepad++, etc.)
  2. Find the STATS object in the script section near the bottom
  3. Update the values:
    • startWeight: Starting weight in pounds
    • currentWeight: Current weight in pounds
    • goalWeight: Goal weight in pounds
    • lastUpdated: Date in YYYY-MM-DD format
    • notes: Any optional notes (plain text)
  4. Save the file
  5. Copy the same values to the STATS object in index.html
  6. Upload both files to your server
